# T-RP
Roleplay system for CSGO in sourcepawn

Usage:
  - Cars:
      - sm_vehicle
  - Inventory:
      - Press INSPECT
  - Job (Apple Harvester)
      - Zone name: Apple_*
  - Job (Miner)
      - Zone name: mining_*
  - Job (Drug Planter)
      - sm_plant
  - Job (Gardener)
      - Zone name: garden_*
  - Job (Garbage Collector)
      - sm_garbagespawns
  - Job (Police)
      - sm_purge
      - sm_helpme
      - sm_criminals
  - Npc Creation
      - sm_npc -- blank npc
      - sm_editnpc -- Aim at npc to edit
  - Money
      - sm_givemoney
      - sm_givebankedmoney
